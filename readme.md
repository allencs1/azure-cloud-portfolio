# Azure Cloud Projects (Beginner → Foundation)

This repository contains hands-on projects demonstrating core cloud concepts using Microsoft Azure.

---

## 📌 Project 1: Virtual Machine (IaaS)

### Overview
I first created a Virtual Machine manually using the Azure Portal to understand the setup process, then configured it with Apache to host a webpage.

### Key Concepts
- Infrastructure as a Service (IaaS)
- SSH access and server management
- Public vs Private IP addressing
- Web server configuration (Apache)

---

## 📌 Project 2: App Service (PaaS)

### Overview
Deployed a Python Flask web application using Azure App Service, removing the need to manage servers.

### Key Concepts
- Platform as a Service (PaaS)
- Application deployment
- Managed infrastructure
- Dynamic responses using query parameters

---

## 📌 Project 3: Azure Functions (Serverless)

### Overview
Created an HTTP-triggered function that responds dynamically based on user input.

### Key Concepts
- Serverless architecture
- Event-driven computing
- HTTP triggers
- Pay-per-execution model

---

## 📌 Project 4: Terraform VM Deployment (IaC)

### Overview
Manually deployed a Virtual Machine using the Azure Portal, then automated the deployment using Terraform from a Linux environment.

### Key Concepts
- Infrastructure as Code (IaC)
- Terraform workflow (init, plan, apply)
- Resource provisioning via code
- Debugging cloud deployment issues
- Virtual networking (VNet, Subnet, Public IP, NIC)
- SSH configuration and access
