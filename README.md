# ☁️ Azure Cloud Portfolio

This repository showcases hands-on Azure projects demonstrating infrastructure deployment, networking, identity management, security, and multi-tier cloud architecture using real-world scenarios.

---

# 📌 Projects

## 🌐 Project 1: Azure Networking & Security Lab

- Configured Azure Virtual Networks (VNets) and subnets
- Deployed multiple virtual machines
- Tested external SSH connectivity
- Tested internal VM-to-VM communication
- Configured Network Security Groups (NSGs)
- Demonstrated traffic allow/deny rules

---

## 🔐 Project 2: Azure IAM / RBAC Access Control

- Created Microsoft Entra ID lab users
- Configured Azure RBAC permissions
- Tested Reader and Virtual Machine Contributor roles
- Validated least privilege access control
- Managed permissions using Azure Cloud Shell

---

## 🏗️ Project 3: Azure Multi-Tier Architecture

- Built frontend and backend Azure architecture
- Implemented subnet segmentation
- Deployed backend VM using Terraform
- Configured private internal communication
- Tested backend services through internal networking
- Demonstrated secure multi-tier cloud design

---

## 🛡️ Project 4: Defense-in-Depth Security

- Secured Azure virtual machine access using NSGs
- Restricted SSH access to a trusted public IP
- Implemented SSH key authentication
- Created custom administrative user
- Validated access control through security testing
- Demonstrated layered cloud security principles
